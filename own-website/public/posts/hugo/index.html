Starting out with Hugo

  1. Download hugo, using pacman in this example. Check official website for other ways.1
1pacman -Sy hugo
  1. Create a new website. Configuration file uses toml by default, yml format is possible with appending -f yml to below command.
1hugo new site my-website
  1. Download a theme and initialize it.
1cd my-website
2git init
3git submodule add git-link-here themes/theme-name
  1. Set the theme on toml file.
1printf "theme = "theme-name"\n" >> config.toml

  1. Change your settings inside config.yml or config.toml.

  2. Check your website, it will be dynamically updated whenever a file is changed.

1hugo server & ; firefox http://localhost:1313/
  1. All post should be located at the folder called content, create a new file.
1hugo new posts/first.md
  1. This file consists three settings, default template could be changed at archetypes/default.md.
1---
2title: "{{ replace .Name "-" " " | title }}"
3date: {{ .Date }}
4draft: true
5---
